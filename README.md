# Adilet Shynggys Graduate Portfolio

Static portfolio website for graduate applications. It is ready for GitHub Pages.

## Deploy on GitHub Pages

1. Create a GitHub repository named `adilet-portfolio` or `username.github.io`.
2. Upload `index.html`, `styles.css`, `script.js`, and this `README.md`.
3. In the repository settings, open **Pages**.
4. Choose **Deploy from a branch**, select `main`, and use the root folder.

The site will publish at the URL GitHub shows in the Pages settings.
